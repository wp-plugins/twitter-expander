== Description ==
  Expand all of your Twitternames @name into a Posting with the correct Link http://twitter.com/name

  [Created by Eric-Oliver M&auml;chler](http://www.maechler.me "Maechler.me")



== Installation ==
  1. Upload des Plugins in den Plugin-Ordner /wp-content/plugins/
  2. In der Plugin-Administration ihres Blogs kann nun Twitter Expander aktiviert werden.
  3. Ab sofort werden alle Twitternamen in einem Beitrag @name mit dem entsprechenden Link erweitert
